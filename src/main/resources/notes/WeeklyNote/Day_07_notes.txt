CREATE TABLE students (
    student_id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    student_name VARCHAR(100),
    student_email VARCHAR(100),
    student_zipcode VARCHAR(10),
    student_phone VARCHAR(20)
);
 
1. Indexing 
	a. Primary Key : 	
	  1. only one unique value is allowed for that column. 
	  2. primary creates the index where we can query directly.
	  3. Student_name = I want to search

	  select * from students where student_id = '12344'
	  will do lookup and give the record back directly.
	  
	b. index on student name: 
	CREATE INDEX idx_students_student_name ON students(student_name);
